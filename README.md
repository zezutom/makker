# makker
_Make_ (formerly _Integromat_) connector. Use the `makker` client (and the server) to manage your low-code and no-code scenarios from your code!

## The Client
Seamlessly plug the `makker` client into your Java or Kotlin code base.

[Check the docs](client/README.md)

## The Server
Not a fan of Java or Kotlin? No problem. Deploy the `makker` server and manage your scenarios through a REST API. 

[Check the docs](server/README.md)
